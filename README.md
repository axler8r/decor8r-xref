# DecoR8R Cross Reference

[`decor8r`][link2-decor8r] is inspired by many `Powerline`-like implementations.
Find links to some of them here.


## Inspiration

The following list points to projects and libraries that influence and inspire
`decor8r`:

1. [Powerline][link2-powerline] — The original `Powerline`.
1. [Oh-My-Bash][link2-ohmybash] — `Powerline` functionality.
1. [Oh-My-Fish][link2-ohmyfish] — `Powerline` functionality.
1. [Oh-My-Posh][link2-ohmyposh2] —  `Powerline` for `PowerShell` version 2.
1. [Oh-My-Posh][link2-ohmyposh3] — `Powerline` for `PowerShell` version 3.
1. [Oh-My-ZSH][link2-ohmyzsh] — `Powerline` functionality.
1. [Posh-Git][link2-poshgit] — The way to disply `git` status!
1. [Powerline-GO][link2-powerlinego] — Powerline `go`.
1. [Starship][link2-starship] — The _Mothership_, when a lot is not enough.
